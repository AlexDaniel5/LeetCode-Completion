Description:
Add the sum of each node, bring a carry of one to the next node if required. Adding through
linked list provides the ability to take the sum of extrememly high numbers; unlike unsigned
integers with a maximum of 4294967295 and double's capacity of fifteen digits.

LeetCode Description:
You are given two non-empty linked lists representing two non-negative integers. The
digits are stored in reverse order, and each of their nodes contains a single digit.
Add the two numbers and return the sum as a linked list.
